# Snake-Game


A GUI app using Tkinter 


## Tech
- Python



## Additional Modules
  - Tkinter
  - Random
  - os
  - sys


## Run Locally

Clone the project

```bash
  git clone https://github.com/shodhanshetty14/Snake-Game-Tkinter.git
```

Go to the project directory

```bash
  cd Snake-Game-Tkinter
```

Install dependencies

```bash
  pip install tkinter

```

Run Program

```bash
  python app.py
```
